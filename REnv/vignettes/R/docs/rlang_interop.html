rlang_interop {REnv} R Documentation

Helper function for R# language interoperability with R language

Description

This function facilitates interoperability between R# and native R by translating R# closure expressions to native R code and executing it.

Usage

rlang_interop(code,
    source = NULL,
    debug = FALSE,
    workdir = NULL,
    print.code = FALSE,
    native.R = 'getOption'('native_rexec'));

Arguments

code

An R# managed closure expression to be converted and executed as native R code.

source

External dependency script file paths required for the execution environment.

debug

Logical indicating whether to debug the translation process (default: FALSE).

workdir

Working directory for executing the R script (default: current directory).

print.code

Logical indicating whether to print the generated R code (default: FALSE).

native.R

Path to custom Rscript executable (default: from options or system default).

Details

This function enables R# code to leverage native R functionality by translating R# expressions to native R code and executing it through Rscript. It handles both Windows and Linux environments automatically.

Authors

xieguigang <xie.guigang@gcmodeller.org>

Value

The result of executing the translated native R code, typically the output of the R script execution.
