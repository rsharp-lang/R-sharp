Rdocuments {REnv} R Documentation

Create html documents for R# package module

Description

Usage

Rdocuments(pkgName,
    outputdir = ./,
    package = NULL);

Arguments

pkgName

the R# package module itself or the package character text name.

outputdir

the output directory path for save the html documents.

package

a character vector for display the package name

Details

the parameter pkgName is different with package parameter, for 1. package parameter is standard for the R# zip package its namespace 2. pkgName parameter usually be one of a clr dll module

Authors

xieguigang

Value

This function has no value returns.
