getJSON {REnv} R Documentation

get json web request

Description

Usage

getJSON(url,
    interval = 3,
    raw.text = FALSE);

Arguments

raw.text

this parameter determine that the function return value is the raw json character text data or R# object value.

url

the web resource of the target json resource object.

Details

this function works on the http_get function, so that the options for http_get: http.cache_dir and http.debug is required as well.

Authors

Value

This function has no value returns.
